# dna-toolkit
A beginner bioinformatics toolkit built in Python — by a biotechnologist learning to code.

## What it does
Converts a DNA sequence into its RNA equivalent by replacing
every Thymine (T) with Uracil (U).

## Usage
```python
from dna_toolkit import transcribe

transcribe("ATGCTA")   # Returns: "AUGCUA"
```
